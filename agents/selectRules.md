# 📈 A股短线交易规则（优化量化版）

> 本规则已全部量化，减少主观判断，提高实战可操作性

---

# 🎯 一、核心原则

1. **量化优先** - 所有标准使用具体数字
2. **三重过滤** - 市场→板块→个股
3. **快速决策** - 5分钟内可完成
4. **风险第一** - 保本比赚钱重要

---

# 📊 二、每日准备工作（8:30-9:00）

## ✅ 必须检查清单

- [ ] 查看昨日涨停家数
- [ ] 查看昨日跌停家数
- [ ] 查看昨日连板成功率
- [ ] 查看今日板块预判
- [ ] 准备10只备选股票池

# 🎯 三、市场情绪判断（决定是否出手）

## 情绪等级定义

| 等级 | 涨停数 | 跌停数 | 连板成功率 | 操作建议 |
|------|--------|--------|-----------|----------|
| **强势** | ≥ 50 | ≤ 5 | ≥ 70% | 积极出击 |
| **分歧** | 30-50 | 5-15 | 50-70% | 控制仓位 |
| **退潮** | ≤ 30 | ≥ 15 | ≤ 50% | 空仓观望 |

## 硬性条件
- 涨停数 < 20 或 跌停数 > 20 → **禁止操作**
- 昨日炸板率 > 30% → **禁止操作**

---

# 🔥 四、板块筛选（决定做哪个方向）

## 板块硬性条件

同时满足：
- 板块涨幅 ≥ 2%
- 板块内涨停 ≥ 2家
- 板块内上涨 ≥ 10家

## 选择标准
- 选择涨幅前三的板块
- 优先选择有政策/新闻催化的板块

---

# 📋 五、竞价选股（9:15-9:25）

## 量化筛选条件

必须**同时满足**以下条件：

| 指标 | 条件 | 说明 |
|------|------|------|
| 竞价涨幅 | 2% ≤ 涨幅 ≤ 6% | 过高诱多，过低弱势 |
| 竞价成交额 | ≥ 5000万 | 保证流动性 |
| 竞价换手率 | ≥ 2% | 有资金关注 |
| 流通市值 | 50亿-300亿 | 避免大盘和小盘 |
| 昨日涨幅 | ≤ 8% | 避免已过热 |
| 昨日换手率 | 5%-20% | 适中的活跃度 |
| 昨日形态 | 非天地板 | 避免高风险股 |

## 排除条件（遇到直接剔除）

- 竞价高开 > 8%（诱多风险）
- 连板 > 5板（退潮风险）
- 昨日天地板
- 有利空公告/减持公告
- 停牌核查/监管关注
- ST股

## 筛选结果处理

- 保留 **5-8只** 最强个股
- 按优先级排序：

```
优先级1：板块龙头（板块涨幅第一）
优先级2：连板龙头（板块内连板数最多）
优先级3：新晋龙头（首次涨停的强势股）
优先级4：老龙头（历史妖股再次启动）
```

| 累计成交量 | | | |
| 是否创新高 | | | |

## 开盘走势量化判断

### 🔥 类型1：强势（可买）

**量化标准**：
- 1分钟内涨幅 ≥ 2%
- 3分钟内创新高
- 前3分钟成交量 ≥ 昨日同时段1.2倍

**操作策略**：
- 优先：打板（涨停价买入）
- 备选：回踩不破竞价低点买入

### ⚖️ 类型2：分歧（最佳买点）

**量化标准**：
- 1-3分钟内涨幅 0-2%
- 最大回撤 ≤ 1%
- 3分钟内收复失地
- 成交量温和放大

**操作策略**：
- **分歧低吸**：在回踩低点附近买入
- 设置止损：低于竞价低点2%

### ❌ 类型3：弱势（放弃）

**量化标准**：
- 1分钟内下跌 > 1%
- 3分钟内最大回撤 > 2%
- 成交量萎缩

**操作策略**：
- 直接放弃该股
- 观察其他备选

## 买入时机

### ✅ 允许买入的情况

1. **分歧低吸**（首选）
   - 冲高后回踩
   - 不破关键支撑位
   - 有量能承接

2. **弱转强打板**
   - 开盘震荡后突然拉升
   - 量能放大2倍以上
   - 直奔涨停

3. **回踩均线**
   - 上升趋势中
   - 回踩MA5/MA10不破
   - 缩量回调

### ❌ 禁止买入的情况

- 竞价直接买入
- 开盘追高大阳线
- 连板 > 3板盲目接力
- 尾盘冲动买入
- 下跌途中抄底

# 💰 七、仓位管理

## 情绪与仓位对应

| 情绪等级 | 单股上限 | 总仓位上限 | 操作策略 |
|---------|---------|-----------|----------|
| **强势** | 30% | 70% | 积极出击，可满仓 |
| **分歧** | 20% | 50% | 控制仓位，半仓操作 |
| **退潮** | 0% | 0% | 空仓观望 |

## 建仓策略

1. **首仓**：不超过计划仓位的50%
2. **加仓**：确认走强后再加
3. **分批**：同一股票最多分2批买入

## 持仓数量

- 强势市场：最多3只
- 分歧市场：最多2只
- 退潮市场：0只

---

# 🛑 八、卖出策略（严格风控）

## 止损规则（必须执行）

| 触发条件 | 操作 |
|---------|------|
| 亏损 ≥ 5% | **立即止损** |
| 跌破关键支撑位 | **立即止损** |
| 1小时内下跌 > 3% | **立即止损** |
| 炸板后无法回封 | **立即止损** |

## 止盈策略

| 盈利幅度 | 操作建议 |
|---------|---------|
| 3%-5% | 观察走势，不强可减半 |
| 5%-10% | 减仓锁定利润 |
| 10%-15% | 继续持有，设置移动止盈 |
| 涨停 | 持有观察，次日竞价决定 |
| 炸板 | 立即减仓或清仓 |

## 移动止盈

- 盈利 > 10% 后，设置止盈线为最高价的 -3%
- 股价跌破止盈线立即卖出

## 危险信号（无条件卖出）

- 放量大阴线（跌幅 > 5%，成交量 > 2倍）
- 龙头断板（炸板无法回封）
- 板块退潮（板块内多数股下跌）
- 跌破重要均线（MA20）
- 出现利空消息
